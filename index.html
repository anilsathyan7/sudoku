A Multiplayer version of classical SUDOKU game

Sudoku is a classical puzzle game, where your objective is to fill the square boxes with numbers:0-9

The following are the additional rues to be followed :-

  • Each 3*3 mini block should contain all numbers: 0-9.
  • Each horizontal row must contain all the numbers: 0-9.
  • Each vertical column must contain all the numbers: 0-9.
  • There exists only one solution.
  • Share this document through url for multiplayer mode.
  • Click 'Finish' when you are done.The game ends when timer expires.
  • Click 'Authorize' to login with your google account for multiplayer mode.
